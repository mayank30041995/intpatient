Good question 👍 — this is where most people get stuck.

You basically need to resize 1 image into multiple widths. I’ll give you 3 easy ways (fastest → pro).

✅ 🔥 Method 1: Easiest (Online – No install)
Use 👉 Squoosh (Google tool)
Open: https://squoosh.app
Upload your image (max-saket-building.webp)
On right side:
Resize → enable
Export 3 versions:
Width	Save as
480	max-saket-building-480.webp
640	max-saket-building-640.webp
720	max-saket-building-720.webp

👉 Keep format = WebP (best compression)

✅ 🔥 Method 2: Photoshop (if you use it)
Open image
Go to Image → Image Size
Change width:
480 → Save As
640 → Save As
720 → Save As

👉 Make sure:

✔ “Maintain aspect ratio” ON
✔ Export → WebP / Quality ~70–80
✅ 🔥 Method 3: Command Line (BEST for devs 🚀)

If you want automation:

Install ImageMagick
brew install imagemagick   # mac
Run:
convert max-saket-building.webp -resize 480 assets/max-saket-building-480.webp
convert max-saket-building.webp -resize 640 assets/max-saket-building-640.webp
convert max-saket-building.webp -resize 720 assets/max-saket-building-720.webp

👉 Done in seconds ⚡
